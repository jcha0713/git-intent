# git-intent

**git-intent** is a Git workflow tool designed for creating [intentional commits](https://intentionalcommits.org/).

## Requirements

- Git (>= 2.0)
- Bun (>= 1.2.14)

## Usage (for testing)

```bash
# To add an intent
bun start add <intention>

# To list all intents
bun start list
```
